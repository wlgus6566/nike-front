<template>
	<div>
	    <SearchInput ref="searchInput" @searchSubmit="searchSubmit" />
	</div>
</template>
import SearchInput from '@/components/search-input';
export default {
    data() {
        return {
            searchKeyword: '',
    },
    components: {
        SearchInput,
    },
    methods: {
        //검색
        searchSubmit(val) {
            this.searchKeyword = val;
            this.initFetchData(); // -> 검색시 api 재호출
        }
    }
};
</script>
  • input은 ELEMENT UI를 사용한다.
  • src>components 디렉토리에 search-input 컴포넌트로 작업되어있다.
  • parent 컴포넌트에 search-input 컴포넌트를 import 시켜 작업한다. 하단의 마크업 참고
