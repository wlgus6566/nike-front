<template>
	<div>
	    <SearchInput ref="searchInput" @searchSubmit="searchSubmit" />
	</div>
</template>
import SearchInput from '@/components/search-input';
export default {
    data() {
        return {
            searchKeyword: '',
    },
    components: {
        SearchInput,
    },
    methods: {
        //검색
        searchSubmit(val) {
            this.searchKeyword = val;
            this.initFetchData(); // -> 검색시 api 재호출
        }
    }
};
</script>
  • input은 ELEMENT UI를 사용한다.
  • src>components 디렉토리에 search-input 컴포넌트로 작업되어 있는 것을 사용한다.
  • parent 컴포넌트에 위와같이 search-input 컴포넌트를 import 시키며 위의 예시를 참고 한다.
