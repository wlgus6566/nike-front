	<span class="select">
		<el-select placeholder="Select" >
			<el-option
				v-for="item in category2Code.listSortOptions"
				:key="item.value"
				:label="item.label"
				:value="item.value"
			>

		</el-select>

<script>
export default {
	data() {
		return {
			category2Code: {
				listSortOptions: [
					{
						value: '',
						label: '대구분',
					},
					{
						value: '11',
						label: '운영비품',
					},
				],
				value: '',
			},
		};
	},
};
</script>
  • select는 ELEMENT UI를 사용한다.
  • 기본 셀렉트 박스는 위와 같이 사용한다.

Filter Select

code


<template>
    <FilterSelect :listSortSelect="category2Code" />
</template>
<script>
import FilterSelect from '@/components/filter-select';
export default {
	data() {
		return {
			category2Code: {
				listSortOptions: [
					{
						value: '',
						label: '대구분',
					},
					{
						value: '',
						label: '운영비품',
					},
				],
				value: '',
			},
		};
	},
	components: {
		FilterSelect
	},
};
</script>
  • src>components 디렉토리에 filter-select 컴포넌트로 작업되어 있는 것을 사용한다.
  • parent 컴포넌트에 위와같이 filter-select 컴포넌트를 import 시키며 위의 예시를 참고 한다.

<template>

	data() {
		return {

                value: [null],
                options: [
                    {
                        value: null,
                        label: '전체 그룹',
                    },
			        {
                        value: 1,
                        label: 'NIKE ADMIN',
						children:[
						    {
		                        value: 2,
		                        label: 'Nike',
		                    },
						]
                    },
                ],
            },
		};
	},
	components: {
		CascaderSelect
	},
};
</script>
  • src>components 디렉토리에 cascader-select 컴포넌트로 작업되어 있는 것을 사용한다.
  • parent 컴포넌트에 위와같이 cascader-select 컴포넌트를 import 시키며 위의 예시를 참고 한다.
