  • ELEMENT UI를 사용한다.
  • 별도의 vue 파일을 만들어 위와 같이 마크업 한다.
  • Modal 파일의 위치는 Parent 파일과 같은 폴더안에 만든다.

Parent File

code


<template>
	<div>
		<button type="button" @click="modalOpen">Modal Open</button>

		<ModalName
			:visible.sync="visible.detailView"
		/>

	</div>
</template>
<script>
import ModalName from '@/경로/ModalName.vue';

export default {
	data() {
		return {
			 visible: {
                detailView: false,
            },
		};
	},
    components: {
       ModalName
    },
	methods: {
		modalOpen() {
            this.visible.detailView = true;
        },
	},
};
</script>
  • Parent Vue 파일에 모달 파일은 Modal.vue을 import 시키며 위와 같이 마크업 한다.
